About this project

The data on this website and used to build the recommender is from www.singletracks.com, obtained through their public API. I extracted the trail data using the API, then used Python (pandas, numpy, sklearn, spacy) to process, analyze and model the data. I used natural language processing (NLP) and clustering techniques on the trail descriptions to extract latent topics describing trails. I used these topics along with additional trail features (length, difficulty, star rating, etc.) to find the similar trails to a given trail to build the recommender. This web app is built using Flask and a Bootstrap starter template.

For more information on this project, you can visit my github project page.
